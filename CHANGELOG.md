# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - Current
### Added
- Initial local-first release.
- Core capabilities: Prompt management (create, search, pin, delete).
- Universal DOM injection into ChatGPT, Claude, and Perplexity via isolated content scripts.
- Support for inline dynamic variables (e.g., `{{topic}}`).
- Built-in Dexie IndexedDB for privacy-focused local storage.
- JSON Import/Export features for vault portability.
